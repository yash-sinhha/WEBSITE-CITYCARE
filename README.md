CityCare Website

CityCare is a healthcare website designed to provide information about medical services, hospital facilities, and patient care. The project focuses on a simple layout, clear structure, and easy navigation.

Features

Responsive website layout  
Clean and simple user interface  
Easy navigation between sections  
Suitable for academic and learning purposes  

Technologies Used

HTML  
CSS  
JavaScript

Getting Started

Clone the repository

git clone https://github.com/yash-sinhha/WEBSITE-CITYCARE.git

Navigate to the project folder

cd WEBSITE-CITYCARE

Open the project

Open the index.html file in any web browser.
